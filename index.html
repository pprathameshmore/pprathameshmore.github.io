FShare — Share files up to 1.5 GB securely, easily

A file-sharing service that allows users to safely and simply share files. Users can upload up to 1.5GB per file, set a download limit, password, or expiry date, and share short URLs. Files are encrypted and stored in Microsoft Azure Blob Storage.

Built with Node.js, Express.js, PostgreSQL, MongoDB, Redis, Docker, OAuth2, Microsoft Azure, React.js and Bootstrap 4 — a hands-on exercise in microservices, with Authentication, File Handling and Short URL split into individual services.
